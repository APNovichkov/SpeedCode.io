# Speedcode API Docs

**Docs are organized by Endpoint**

## /algorithms

> GET

Returns all algorithms available for a specific programming language

#### Input Params

```
{
    'language': 'java'
}
```

```
{
    'id': 'id',
    'name': 'Bubble Sort',
    'category': 'Sorting',
    'description': 'Iterative Sorting Algorithm',
    'difficulty': 3 (int),
    'attempts:' 5 (int)
}
```
